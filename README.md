# YAFIT

YAFIT (**Y**et **a**nother **f**eedback **i**nformation **t**ool) ist ein Tool für Lehrkräfte um von ihren Schüler:innen anonym Feedback zu erhalten.

YAFIT ist eine WPF-App und setzt mindestens Windows 10 vorraus.
